Project Title:
SecureNet Crypto System – Cryptography & Network Security Learning System

Description:
This project is a web-based application that demonstrates various cryptographic algorithms and their working. It includes encryption, decryption, flow visualization, file encryption, and history tracking.

Technologies Used:
- HTML
- CSS
- JavaScript
- Node.js (Backend)
- Express.js

Algorithms Implemented:
- Caesar Cipher
- Substitution Cipher
- AES (Advanced Encryption Standard)
- DES (Data Encryption Standard)
- Hill Cipher
- Diffie-Hellman Key Exchange
- Blowfish (Conceptual)

Features:
- Encryption and Decryption Tool
- File Encryption
- Download Encrypted Output
- Encryption & Decryption Flow Diagrams
- Dark Mode UI
- History Storage (Backend)
- Export History as JSON
- Clear History Option

How to Run:
1. Install Node.js
2. Open terminal in project folder
3. Run:
   npm init -y
   npm install express cors
   node server.js
4. Open index.html in browser

Future Enhancements:
- Database integration (MongoDB)
- User login system
- Real-time encryption visualization
- Deployment to cloud

Conclusion:
This project helps understand practical implementation of cryptographic techniques and network security concepts.
